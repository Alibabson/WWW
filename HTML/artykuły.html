Początki firmy

14 czerwca 2025

Wstępną ideą założyciela - Eryka - było połączenie pasji informatycznej oraz muzycznej. Pierwszymi krokami było tworzenie kodu który w pewnym stopniu nawiązuje do muzycznych marzeń:

Przykład jednego z pierwszych kodów: 


            using System;
            using System.Media;
                
            class Program
            {
                static void Main()
                {
                    Console.Beep(440, 500); 
                    Console.WriteLine("Proszę bardzo!");
                }
            }

Po uruchomieniu tego programu konsola odtwarzała prosty dźwięk z podaną częstotliwością.
